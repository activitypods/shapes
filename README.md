# ActivityPods shapes repository

A repository with shapes and shape trees used by [ActivityPods](https://activitypods.org) applications.

## Running locally

```bash
yarn install
yarn start
```

## Submitting shapes

Feel free to open a PR to submit new shapes and shapetrees.
